# tunz - android app to play ma tunes and junk

you'll need to give it the permission it DOESN'T ask for - sorry

my Music dir has 4 dirs

An  (annie - my gf's songs - ones i don't like)
St  (steve - my songs - ones she doesn't like)
_a  (a songs that we both like and could listen to repeatedly)
_b  (b songs that we both like but once max please)

shuf will order em randomly MINUS the songs you already heard
   (upon hearing all of em, the done list is killed)

check shuf or not
check the dirs you want to hear
play/pause is the "action button whatev"

make sure to name your songs as
group - year album song# - title.mp3

or it'll sort wrong etc
