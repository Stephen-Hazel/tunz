# tunz - android app to play ma tunes and junk

hopefully it asks for permissions properly now?

Music/tunz dir has pickable subdirs under it.  Mine are...

- An  (annie - my gf's songs - ones i don't like)
- St  (steve - my songs - ones she doesn't like)
- _a  (a songs that we both like and could listen to repeatedly)
- _b  (b songs that we both like but once max please)

Keep your dir names short or they won't fit on the ONE row alotted.

Shuf will order em randomly MINUS the songs you already heard
- Via a done list stored across sessions.
- Interleaving songs from each picked dir.
- Upon hearing all of em, the done list is killed.

check shuf or not
check the dirs you want to hear

play/pause is the bottom right "action button whatev thingo"

make sure to name your songs as:  group - year album song# - title.mp3
- the middle part is optional but
- there must be - - in the filename or it'll sort wrong etc

good luck
